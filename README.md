# VawSafe
> To run on local machine:

1) Open project folder with VSCode

> Inside Terminal:

2) `py -m venv .venv` 
3) `cd vawsafe`
4) `python -m pip install -r requirements.txt`
5) `python manage.py install tailwind`
6) `python manage.py tailwind start`

> In another Terminal:

7) `py manage.py runserver`
